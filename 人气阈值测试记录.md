# 人气阈值和持仓数量组合测试

## 测试配置
- **人气阈值**: 前10名、前20名、前30名
- **持仓数量**: 1-5只
- **每只资金**: 100%、50%、33.33%、25%、20%

## 测试命令

### 前10名人气股测试
```bash
# 1只持仓 - 100%资金
python scripts/backtest_hot_rank_rise2_strategy.py --config config/strategies/hot_rank_rise2.yaml --param.hot_top_n 10

# 2只持仓 - 50%资金 (需要先修改代码 max_positions=2, nominal_cash=0.5)
python scripts/backtest_hot_rank_rise2_strategy.py --config config/strategies/hot_rank_rise2.yaml --param.hot_top_n 10

# 以此类推...
```

### 前20名人气股测试
```bash
python scripts/backtest_hot_rank_rise2_strategy.py --config config/strategies/hot_rank_rise2.yaml --param.hot_top_n 20
```

### 前30名人气股测试
```bash
python scripts/backtest_hot_rank_rise2_strategy.py --config config/strategies/hot_rank_rise2.yaml --param.hot_top_n 30
```
